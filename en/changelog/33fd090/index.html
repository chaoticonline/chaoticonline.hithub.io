Published Content-related changes

  • Revised all Casts added so far, removing some names of people/creatures that hasn’t actually appeared, reordering and completing others

  • With the addition of the new Shortcode Template expr, the published Articles about Heptadd and about Tartarek, the only ones with identification of their names so far, have been updated, replacing the usual bold Markdown used for these expressions with the new Shortcode definition

Fixed

Hugo Templates-related changes
  • Fixed an issue with the GDPR Cookie routine in which, given the unspecified expiration date, the Cookie was removed every time the browser in which it was created was closed. By all means this represents that all efforts made in previous revisions to keep the Cookie, so in future visits the User wouldn’t be asked to agree with the Terms of Use once again, were in vain

  • Fixed a wrong keyword in Episode #22, Season #01 pointing to Mallash while the right Creature appearing right at the beginning is Valanii Levaan

  • Fixed a URL in the published Article about the Lava Pond so it now properly points to Kiru Village (Brazilian Portuguese only) and two other pointing to the Episodes Over-Under Rent Asunder and Everything’s in Flux (both languages)
