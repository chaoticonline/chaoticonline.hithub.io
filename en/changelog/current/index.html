Added

Structure-related changes

  • Added a missing i18n entry for Tangath Toborn, In Training

  • Added a Content File to ease out the development of a 404 Page, even though, for now, after compiled, the resulting has to be manually renamed and moved one level above

  • Added the Hugo Debugprint Project to the main theme, conditioning its CSS file to the development environment

Removed

Structure-related changes

  • With the massive change in the Content Files directory structure (see below), all localized files under /content have been moved one level deeper accordingly to the available language, which, by all means, represents files removal and the inclusion of a new ones

  • Removed the FrontMatter Parameters target and language since they’re not needed anymore

  • Removed the outer glow when clicking the dropdown menus of Cards Search Tool Page

Changed

Structure-related changes

  • Recreated the whole Content Files’ directory structure following the multi-language model /content/<LANG>/<SECTION>/file.md just like we do with Data Files, i18n entries and static files specific to a language, which, by all means, represents files removal and the inclusion of a new ones

  • With the restructuring above, ALL Content Files have been updated in order to:

    1. Use a slug instead of a Custom URL This change was made because while shorter URLs were preferable in past, looks like nowadays verbose and well descriptive URLs are what a website should have <_<
    2. Create a relation between Content Files across multiple languages through Hugo’s predefined FrontMatter Parameter translationKey because if any directory between /content and the file name diverge from the default language’s structure (English) the relationship is lost and then Hugo is unable to synch translations “automagically”
    3. Update ALL links in all published Articles and Episodes Pages
    4. Remove unnecessary FrontMatter Parameters (see above)

  • With the inclusion of a Content File for the development of the 404 Page (see above), a file .gitignore has been added to prevent the /404 directory from getting versioned (and published to GitHub Pages)

  • Redone the way the Navigation Menu is generated by building it directly in the config.yaml for Hugo to render it instead of adding each entry manually

Hugo Templates-related changes

  • Improved the palliative solution to keep all Bootstrap Cards’ with the same size

  • Changed the way Hugo’s Paginator is configured between the Changelog Page and everything else because with the new multi-language structure relying on .Section for such distinguishing doesn’t seem to be enough, even though we expand the condition to take into account all available languages

  • Changed the Taxonomy Terms Template to not include in its listing the Changelog Main Page because with the new multi-language structure this specific Page makes use of Hugo’s predefined FrontMatter Parameter type which conflict with our own Taxonomy Term reserved to Cards Types

  • Improved the Taxonomy Terms Template not only to follow the “standard” of the other Templates by making use of the Base Template but also to not have an Advertisement Block on it, which indirectly improves the performance of the Cards’ Search Tool

Public Content-related changes

  • Improved the display of Targubaj’s Big Picture, focusing more on the top part rather than in the bottom

  • Now that any observation regarding one or more Voice Actors of a Creature breaks to a new line, it was possible to complement the notes in the Data File about Cerbie, specifying that Left, Middle and Right refers to each one of his heads (Brazilian Portuguese only)

  • Updated Changelog Page

CSS-related changes

  • Improved the positioning of Quotes’ Authors

  • Improved margins in Taxonomy Terms Page

  • Improved texts’ vertical alignment of dropdown menus of Cards Search Tool Page

  • Fixed the Image Credits of Milla’iin’s and Erak’tabb’s Big Pictures
