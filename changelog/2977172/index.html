Chaotic Changelog rev. 2977172

Changelog

  rev. 2977172

 

Added

Structure-related changes

  • Applied Content File Componentization to the Changelog Page with i18n entries and additional CSS/JS files to hide most of the contents in a collapsible block

  • Added our Github Account as another value of sameAs Property for Schema.org’s Organization Entity

  • Added Content Files, Data Files and i18n entries for the following Pages as a counterpart of the references made in the Article about Iron Pillar published in this revision: Gigantroper (Battlegear) and Ice Pillar, Jade Pillar, Jagged Pillar, Steam Pillar, Stone Pillar and Wooden Pillar (Locations)

  • Added an even better way to display the expanded text of <abbr> tags on mobile

Hugo Templates-related changes
  • Added the HTML attribute nofollow to the Terms of Use and Changelog pages in an attempt of not having Google considering them as most linked data-source just because they appear in all pages when the important thing is to have interconnected Pages (i.e. Articles linking to an Episode)
Public Content-related changes

  • Added keywords to the following published componentizable Articles: Maxxor, Protector of Perim; Najarin, Fluidmorphers’ Foe; Najarin, High Muge of the Lake; and Najarin, Younger

  • Added missing FrontMatter Parameter datePublished to the following published Articles/Pages: Bodal; Bodal, Flamedrill Researcher; Maxxor, Protector of Perim; Najarin, Fluidmorphers’ Foe; Najarin, High Muge of the Lake; Najarin, Younger; Magmon; Magmon, Retaliator; Magmon, Engulfed; Takinom, the Shadowknight; Zhade; Wamma; Wamma, Hive Ordnance; Wamma, Flame Guard; and Changelog

  • Published Article about the Iron Pillar

Removed

Structure-related changes
  • With Hugo’s Asset Minification implemented in the previous revision, there’s not need to ignore anything and, therefore, the .gitignore file has been removed

Changed

Structure-related changes

  • Changed the i18n identifiers of Cards’ Pages Headlines to be more consistent with the revamped SEO schema

  • Updated Bing and Yandex access codes

Hugo Templates-related changes

  • Modified the Include Shortcode to not ask for a file if the request format is Git Hash, “throwing an error” if GIT HEAD doesn’t exist because the repository is corrupted or there’s no repository at all. This because the only purpose of this feature, as for now, it to retrieve the previous commit’s hash to fill in the Changelog Page’s Section Headings

  • Modified the Navigation Partial Template to let the Language Switcher column’s width to be handled by Bootstrap, which allowed us to add some spacing between the Site Logo the main Navigation Menu

  • Revamped the SEO implementation of, initially, all Cards’ Pages by having a few componentizable small Partial Templates for each directive, provider greater maintainability to the code

Public Content-related changes

  • Wording corrections and additions and links and keywords review for the published Articles about Maxxor and Najarin

  • Renamed the Brazilian Portuguese version of Castle Pillar to Pilar do Castelo instead of Castelo dos Pilares because, even though it differs from what’s has been used in the Brazilian Portuguese Audio (debuting in Episode #22, Season #01), it sounds more correct, considering Castle Pillar is one of the seven Pillars holding the ceiling up in the UnderWorld, instead of a Castle made of Pillars

  • With the renaming of the Brazilian Portuguese version of Castle Pillar, all occurrences of the old name have been changed throughout the site

  • Updated Changelog Page

CSS-related changes
  • Increased a little bit the distance of the Language Switcher from the right border because the text of the active language was getting briefly cut-out during the collapsing animation

Fixed

Structure-related changes

  • Fixed an issue with the new combined CSS file from individual componentizable sources introduced in previous revision that included the font family file exclusively used by Cards Search Tool in all other files because the required rules were duplicated in the main font family used (Quicksand)

  • Fixed an issue with Structured Data for Videos in which the Episode Thumbnails weren’t being added because the corresponding structure wasn’t updated when we moved a good part of FrontMatter Parameters to Hugo’s Data Files

Hugo Templates-related changes

  • Fixed some syntax “errors” in List and Single Page Templates and Breadcrumbs Partial Template which, even though used to work flawlessly with version Hugo v0.47 and the resource provided by the modification is not even used, it does not work with the upgraded Hugo version using the updated version of GO

  • Fixed an issue that was preventing Videos Listing Pages to have a thumbnail

Public Content-related changes
  • Fixed all Content Files of the Trivias so the fix applied to bring back the Summaries applied a few revisions ago could work on them as well
CSS-related changes
  • Fixed the collapsible Navigation Menu on mobile devices that was getting all messed up after the changes made in previous revision

Image-related Changes

  • Added a smaller version of site logo to be used in SEO Structured Data. It doesn’t look good, but if Google owns the Internet and it says so…

  • Added the following Big Pictures referenced in the Article about Iron Pillar published in this revision: Jade Pillar, Stone Pillar and Wooden Pillar (Locations)

  • Added the Card Image for Iron Pillar as a complement for its Article published in this revision

Miscellaneous changes

  • Updated Sitemap Files for Battlegears, Locations and Index

  • Updated minified CSS and JS
